```cpp
두 개의 큰 수를 조작하는 에디터 프로그램을 작성하려고 한다.

에디터는 다음의 기능들을 수행할 수 있어야 한다.
1. 에디터는 두 줄을 제공하며, 각 줄마다 하나의 수가 쓰여진다.
2. 에디터에 적혀있는 수의 왼쪽 또는 오른쪽에 다른 수를 붙여 쓸 수 있다.
3. 에디터에 적혀있는 두 개의 수를 줄 순서대로 연결하여 하나의 수로 만들었을 때, 그 수에서 특정 수의 등장 횟수를 셀 수 있어야 한다.

에디터는 두 줄을 제공하며, 각 줄마다 쓰여질 큰 수가 주어진다.
주어지는 수는 1부터 9까지의 숫자들의 배열로 이루어져 있다.
배열에 주어진 숫자들을 연결하여 하나의 수를 완성한다.
아래는 배열로 주어진 숫자들을 이용하여 만들어지는 수를 보여준다.
[5, 4, 3, 1, 9, 5, 5, 1, 1, 1] -> 5431955111
[1, 1, 1, 2, 1, 2, 1, 2, 1, 2, 4, 4, 3, 1, 1] -> 111212121244311
초기에 에디터의 각 줄에 주어진 수들이 순서대로 쓰여진다.

에디터에 적혀있는 수의 왼쪽 또는 오른쪽에 다른 수를 붙여 쓸 수 있다.
5431955111의 왼쪽에 215를 붙여 쓰면 2155431955111가 된다.
5431955111의 오른쪽에 215를 붙여 쓰면 5431955111215가 된다.

에디터에 적혀있는 두 개의 수를 줄 순서대로 연결하여 하나의 수로 만들었을 때, 그 수에서 특정 수의 등장 횟수를 셀 수 있어야 한다.
이 때 특정 수는 3자리 수로 주어진다.

에디터에 5431955111와 111212121244311가 순서대로 쓰여있는 경우
연결하여 만들어지는 수는 5431955111111212121244311이 된다.
```
