# C++ 단축키

## 빌드
Ctrl + F7 : 현재파일 컴파일

Ctrl + Break : 빌드 중지

## 소스
Ctrl + K, C : Comment In

Ctrl + K, U : Comment Out

## Ctrl + Shift + F : Find in File
%참고% Find를 Increment Search(증분검색) 로 바꾸면 좀더 편리함

VI에디터처럼 착착 검색을함. UI 가 따로뜨지않아 편리함.

## 특정함수나 변수위치로 이동
Ctrl + F12

## 이전위치 이동
Ctrl + - : 이전 클릭한 소스 위치로 이동 ( 검색 같은거 할경우에 유용 )

Ctrl + Shift + - : 역방향 소스 위치 이동

## Debug모드
Ctrl + F10 : 한단계씩 실행.

Ctrl + F11 : 가장 세부적으로 실행.
